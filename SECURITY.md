# Security Policy

## Supported versions

No released version is currently supported. This table will be updated before `0.1.0-rc.0`.

## Reporting a vulnerability

Use GitHub private vulnerability reporting for this repository. Do not include access tokens,
prompts, user messages, credentials, or opaque compaction payloads in a public issue.

Reports should include the affected version, operating system, reproduction steps, and the impact.
The maintainer will acknowledge a complete report within seven days.
